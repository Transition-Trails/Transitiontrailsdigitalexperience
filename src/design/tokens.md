# Design Tokens

The design system is based on the following token categories:

## Colors
- Primary
- Secondary
- Accent
- Neutral (Grays)

## Typography
- Headings (H1-H6)
- Body (Regular, Bold)
- Labels

## Spacing
- 8pt grid system (space-1 = 4px, space-2 = 8px, etc.)

## Tokens File
See `src/frontend/src/styles/tokens.css` for implementation.
