# Getting Started

## Prerequisites
- Node.js v20+
- npm v10+
- Git

## Setup
1. Clone the repo.
2. Run `npm install` in `src/frontend`.
3. Run `npm run dev`.

## Key Concepts
- **Tokens**: We use CSS variables for theming. See `src/frontend/src/styles/tokens.css`.
- **CMS**: Look for comments `// CMS content here` to identify dynamic regions.
